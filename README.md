# Wall Calendar App
Project URL: https://interactivewallcalendar.netlify.app/

Designed an interactive **Interactive WallCalendar** using React.
It allows users to manage notes, events, and daily activities with a visually aesthetic interface inspired by a real hanging calendar.

---

## Features

### Calendar UI

* Monthly calendar view
* Smooth page flip animation
* Month-specific themes
* Notes section shifts below calendar on small screens

### Notes System

* Add notes to any date
* Edit and delete notes
* Sidebar view for all saved notes

### Events Management

* Add events with:
  * Date range
  * Weekly repetition
* Event indicators on calendar
* Delete events anytime

### Marking-off Dates

* Mark completed days
* Toggle cross/uncross
* Visual strike-through effect

### Holidays

* Predefined holiday highlights
* Special indicator on calendar

---

## Tech Stack

* **Frontend:** React (Hooks)
* **Styling:** Tailwind CSS
* **Storage:** LocalStorage
* **Deployment:** Netlify

---

## How to Run

1. Clone the repository:
   git clone https://github.com/Hanvitha-1464/Calendar.git

2. Navigate to the project:
   cd Frontend

3. Start the app:
   npm run dev
